Probablistic Mortality Prediction

This is calculator is the deployment of the model produced as a result of the analysis of data from 127,134 adult emergency laparotomies included in the National Emergency Lapatomy Audit (NELA). The code for our analysis is avilable on GitHub under MIT License

The work has been submitted to a journal but has NOT yet been peer reviewed and the model is yet to recieve a CE mark. Therefore the form and API offered here should currently be used ONLY for educational and research purposes and should NOT be used to guide patient care.

Interactive form

An interactive form is available to use our novel model. Care should be taken to ensure the correct data is filled in, paying particular attention to units.

Model Overview

The model is uncertainty aware model of mortality after emergency lapraotomy. Lactate and Albumin, if left blank, are imputed using dedicated imputation Generalised Additive Models (GAMs). All other fields are required. The final probabilty distribtion comprises 1000 samples from a final mortality GAM
